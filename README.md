# Nuxt Static

## Installation

Using yarn:

```
yarn add @nuxt/static
```

Using npm:

```
npm i @nuxt/static
```

## Usage

```
npx nuxt static
```

This command will (re-)build project if necessary and use `nuxt generate` or `nuxt export` based on `target` option set in `nuxt.config`.
